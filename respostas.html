Respostas

1.A WEB é uma das ferramentas de acesso à internet e a Internet é a rede de conexão dos computadores que estão interligados.

2. O que acontece é que digitamos o endereço do site (URL) no navegador web, o navegador faz a busca do IP para o nome do domínio, solicita uma requisição HTTP para o servidor web. Ele responde com um redirecionamento permanente ao domínio solicitado e o navegador segue o redirecionamento enviado através do servidor WEB.

3. Uma Web Site não é considerada aplicação Web por não necessitar de estar hospedada em um domíno WEB vinculado a um host.

4. HTML não é uma ligangem de programação, é uma linguagem utilizada para marcação.
